# scpcli — Claude Code Instructions

Go CLI for the Samsung SDS Cloud Platform (SCP) OpenAPI v1. Module: `github.com/berryp/scpcli`. Does not work with SCP v2.

## Hard Rules

- **Never edit `*_gen.go` files.** They are fully generated by `tools/codegen`. Changes will be overwritten.
- **Never add a typed client layer.** All generated commands call `RunRequest` directly. The OpenAPI spec has type mismatches (fields declared `string` that return objects) — raw byte pass-through is intentional.
- **Never move global flags into Kong.** `--version`, `--output-file`, `--filter`, `--print`, `--completion` are pre-parsed from `os.Args` in `parseGlobals()` before Kong is constructed. Kong panics if the same flag name appears at two levels; API operations reuse names like `version` and `filter`.
- **Fix spec problems in `openapi.yaml`, not in Go code.** Re-run `mise run codegen` after any spec change.

## Architecture

```
cmd/scpcli/main.go          entry point → commands.Run()
internal/
  auth/                     HMAC-SHA256 signing
  client/client.go          HTTP client: Do, BuildRequest, Get
  config/                   config.Load() — ~/.scp/*.json + env overlay
  version/version.go        version string (set via ldflags at build time)
  commands/
    root.go                 Kong setup, parseGlobals(), RunRequest, withClient()
    filter.go               --filter (yq, $ENV support)
    print.go                --print (curl/python codegen)
    completion.go           --completion (fish/bash)
    util.go                 PrintJSON, resolveBody
    cli_gen.go              top-level CLI struct (generated)
    *_gen.go                one file per service tag (generated)
tools/
  codegen/                  reads openapi.yaml → writes internal/commands/*_gen.go
  scraper/                  scrapes SCP portal → writes openapi.yaml
openapi.yaml                source of truth for all operations
```

Hand-written files: everything except `*_gen.go` and `cli_gen.go`.

## Key Invariants

**`RunRequest` signature** (`internal/commands/root.go`):
```go
func RunRequest(method, path string, pathParams, queryParams map[string]string, body []byte) error
```
Generated commands call this directly. Do not change the signature without regenerating all `*_gen.go` files.

**Version injection** — `internal/version.version` is a package-level `var` set by `-ldflags`. GoReleaser injects `{{ .Version }}` at release time. Local builds get `"dev"`.

**Config precedence** — env vars override `~/.scp/config.json` / `~/.scp/credentials.json`. See `internal/config/` for the full list of `SCP_*` env vars.
